Syntax Reference

note

Synopsis

note QUANTIZE|STEP [OPTION]... NOTE BLOCK [, note ...]

basics

QUANTIZE
note 1/4 {
  C D E ~
}
STEP
note 480 {
  C D E ~
}
NOTE BLOCK
note 1/8 {
  C1  // C, octave is 1
  C#2 // C sharp, octave is 2
  D>  // D, octave is temporary 3
  D<< // D, octave is temporary 0
  Bb  // B flat, octave is last value (in this case 2) if not specified
  ~   // extend prev note with specified QUANTIZE or STEP length
  ^   // rest for specified QUANTIZE or STEP length
  C,E,G // C chord
}

options

from
note 1/4 from 1920 { C } // by step
note 1/4 from m2 { C }   // by measure
note 1/4 from b2 { C }   // by beat
note 1/4 from 2:1 { C }  // by measure and beat
note 1/4 from 2:1:120 { C } // by measure, beat and tick

with other note

note 1/4 {
  C D E ~
},
note 1/4 {
  B B B B
}
